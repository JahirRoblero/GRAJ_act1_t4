# GRAJact1_t4

Actividad 1 de introducción a Spring Boot.

## Autor

Ángel Jahir Gómez Roblero

## Descripción

Proyecto creado con Spring Boot y Gradle. Contiene controladores y endpoints que muestran información básica del estudiante y sus materias.

## Tecnologías

- Java 21
- Spring Boot
- Spring Web
- Gradle
- HTML, CSS y JavaScript

## Endpoints

- `/api/proyecto` - Muestra el nombre del proyecto.
- `/api/nombrecompleto` - Muestra mi nombre completo.
- `/api/edad` - Muestra mi edad.
- `/api/mismaterias` - Muestra una lista de materias.
